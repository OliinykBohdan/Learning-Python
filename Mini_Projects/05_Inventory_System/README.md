# Inventory System (Python)

## About
Simple console-based Inventory System application written in Python.
This project was built to practice basic OOP concepts and object interaction.

## Features
- Add a product
- Remove a product
- Show all products in stock

## How to run
```bash
git clone https://github.com/OliinykBohdan/Learning-Python.git
cd Learning-Python/Mini_Projects/05_Inventory_System
python Inventory_System.py
```

## Concepts
- OOP (classes and objects)
- Object interaction
- Working with lists of objects
