# Marketplace Lite (Python)

## About
Simple console-based Marketplace Lite written in Python.
This project was built to practice basic OOP concepts and object interaction.

## Features
- Add products to store
- Add products to cart
- Remove products from cart
- Show store products
- Show cart contents

## How to run
```bash
git clone https://github.com/OliinykBohdan/Learning-Python.git
cd Learning-Python/Mini_Projects/11_Marketplace_Lite
python Marketplace_Lite.py
```

## Concepts
- OOP (classes and objects)
- Object interaction
- Working with collections of objects
