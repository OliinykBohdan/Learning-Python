# RPG Duel (Python)

## About
Simple console-based RPG Duel written in Python.
This project was built to practice basic OOP concepts and object interaction.

## Features
- Add items to inventory
- Remove items from inventory
- Show character stats
- Simulate character battle

## How to run
```bash
git clone https://github.com/OliinykBohdan/Learning-Python.git
cd Learning-Python/Mini_Projects/10_RPG_Duel
python RPG_Duel.py
```

## Concepts
- OOP (classes and objects)
- Object interaction
- Working with lists of objects
