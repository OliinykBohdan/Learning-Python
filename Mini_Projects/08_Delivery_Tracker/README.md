# Delivery Tracker (Python)

## About
Simple console-based Delivery Tracker application written in Python.
This project was built to practice basic OOP concepts and object interaction.

## Features
- Add parcel
- Update parcel status
- Show all parcels

## How to run
```bash
git clone https://github.com/OliinykBohdan/Learning-Python.git
cd Learning-Python/Mini_Projects/08_Delivery_Tracker
python Delivery_Tracker.py
```

## Concepts
- OOP (classes and objects)
- Object interaction
- Working with lists of objects
