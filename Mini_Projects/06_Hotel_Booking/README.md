# Hotel Booking (Python)

## About
Simple console-based Hotel Booking application written in Python.
This project was built to practice basic OOP concepts and object interaction.

## Features
- Add room
- Check-in guests
- Check-out guests
- Show room status

## How to run
```bash
git clone https://github.com/OliinykBohdan/Learning-Python.git
cd Learning-Python/Mini_Projects/06_Hotel_Booking
python Hotel_Booking.py
```

## Concepts
- OOP (classes and objects)
- Object interaction
- Working with lists of objects
