# Quiz App (Python)

## About
Simple console-based Quiz application written in Python.
This project was built to practice basic OOP concepts and object interaction.

## Features
- Ask question
- Check answer
- Show results

## How to run
```bash
git clone https://github.com/OliinykBohdan/Learning-Python.git
cd Learning-Python/Mini_Projects/07_Quiz_App
python Quiz_App.py
```

## Concepts

- Object interaction
