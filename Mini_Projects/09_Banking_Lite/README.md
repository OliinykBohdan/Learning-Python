# Banking Lite (Python)

## About
Simple console-based Banking application written in Python.
This project was built to practice basic OOP concepts and object interaction.

## Features
- Show balance
- Make transfer
- Show history

## How to run
```bash
git clone https://github.com/OliinykBohdan/Learning-Python.git
cd Learning-Python/Mini_Projects/09_Banking_Lite
python Banking_Lite.py
```

## Concepts
- OOP (classes and objects)
- Object interaction
