# Basics

This folder contains my Python basics practice and learning materials.

Here I practice core Python concepts through:
- small exercises
- syntax examples
- mini tasks
- simple logic implementations

The goal of this section is to build a strong Python foundation before moving to larger projects.

---

## Topics Covered

### Python Basics
- variables
- conditions (`if`, `elif`, `else`)
- loops (`while`, `for`)
- `break` / `continue`
- functions
- modules

### Data Structures
- lists
- tuples
- sets
- dictionaries

### File Handling
- reading and writing files

### Functional Programming
- closures
- decorators

### Object-Oriented Programming (OOP)
- classes and objects
- constructors
- encapsulation
- inheritance
- polymorphism
- abstraction
